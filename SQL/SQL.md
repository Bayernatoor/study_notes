## What is SQL

- Structure Query Language. 
- used to interact with and manage relational databases
- Can be used to perform operations such as: Create, Reading, Updating, Deleting records (CRUD) within a DB 

## What is NoSQL

- Not ONLY SQL 
- Used in a DB that does not use SQL. Each NoSQL DB typically has it's own querying language.
- Ex: MongoDB uses MQL (MongoDB Query Language) and elasticsearch uses a JSON API.


1. NoSQL DBs are usually non-relational
2. NoSQL have a variety of storage methods such as: Key-value, graph, wide-column etc..
3. SQL DBs usually have a defined schema 
4. SQL DBs are table-based
